## Установка

```
composer install
```

## Настройка

Создать файл `.env` на основе `.env.example`

Распаковать содержимое архива ВК в каталог `archive-vk`
Распаковать содержимое архива Телеграм в каталог `archive-telegram`

## Запуск

Запустить
```
php parse
```

И следовать инструкциям в командной строке
